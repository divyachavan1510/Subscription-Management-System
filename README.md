Subscription Management System

A Java-based application to efficiently manage users, services, and subscriptions. The system is built using JDBC and MySQL, supporting complete CRUD operations with a focus on clean architecture and reliability.

 Features:
User, Service, and Subscription management
Create, Read, Update, Delete (CRUD) operations
Tracks active and inactive subscriptions
Structured using OOP principles
Exception handling and stable database connectivity

Tech Stack:
Java
JDBC
MySQL
OOP (Object-Oriented Programming)

 Project Structure:
model → Contains entity classes
dao → Database operations (CRUD logic)
service → Business logic
util → Database connection handling

 How to Run:

Clone the repository
git clone https://github.com/divyachavan1510/Subscription-Management-System.git
Open in IntelliJ IDEA / Eclipse
Configure MySQL database
Update DB credentials in connection file
Run the main class
